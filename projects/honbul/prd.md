## 개요
한국 신화 삼계(명부·인간계·천계)를 무대로 한 벽돌깨기 + 덱빌딩 + 로그라이크 모바일 게임. 윤회의 굴레에 갇힌 혼이 새 그릇에 들어가 결계를 깨고 환생을 얻는 30분 던전 클라이밍. Pre-production / Design Draft v0.2 — 핵심 설계 확정(엔진·아트·테마·F2P 정책·컨트롤). 영감: 타이탄 브레이커·Slay the Spire·Brotato·Peggle. 워크플로우: Claude 하네스 + Codex CLI 실행.
