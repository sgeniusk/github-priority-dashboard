# 해치의 데일리 브리핑 — PRD

> ⚙️ /sync-project 미실행 스캐폴드 — `projects.json` 데이터에서 자동 생성. 실제 문서를 반영하려면 `/sync-project cmds-daily-briefing`를 실행하세요.

## 개요
해치가 CMDS 1일 브리핑을 날짜별 웹 페이지로 렌더링하는 Vercel 앱. 백상현 daily page에서 영감을 받은 에디토리얼형 UI, `/daily/YYYY-MM-DD` 라우트. 카드별 검토 액션을 Hermes action queue로 넘기는 통합 진행 중. Production URL 가동.

## 기술 스택
- Next.js
- Vercel
- TypeScript
- Hermes action queue

## 핵심 기능
- (PRD의 핵심 기능을 /sync-project로 채우기)
